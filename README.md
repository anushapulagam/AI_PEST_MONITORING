# 🌾 AI Pest Monitoring System

An AI-powered pest detection system that identifies agricultural pests from images and provides instant treatment recommendations to farmers.

## 🔗 Live Demo
👉 [Try it on HuggingFace Spaces](https://huggingface.co/spaces/anushapulagam/AI_Pest_Monitoring)

## 🐛 What It Does
- Upload a pest image
- AI model detects the pest type
- Shows problem description
- Provides step-by-step treatment solutions

## 🐞 Detectable Pests (18 Classes)
Ants | Aphids | Armyworm | Bees | Beetle | Bollworm | Caterpillar | Earthworms | Earwig | Grasshopper | Mites | Moth | Sawfly | Slug | Snail | Stem Borer | Wasp | Weevil

## 🛠️ Tech Stack
- TensorFlow / Keras (CNN)
- Flask (Backend)
- HTML, CSS, JavaScript (Frontend)
- HuggingFace Spaces (Deployment)
